Billing and Scheduling



This is a website and database I built from scratch and maintain for Brownstone Tutors using Django. It is used by about 100 tutors daily to schedule their sessions and used by the administration to process about 100 client invoices per month. One of the main features is a custom Google Calendar engine clone using fullcalendar.js and a custom user profile engine using OAuth2.
